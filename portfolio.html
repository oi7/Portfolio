Tanks (2.5D Multi-player Game)

Completed February 2016
Individual This is a Unity tutorial project from Unite Boston 2015

Tanks is a 2 player, one keyboard shooter game. It is a Tank vs Tank battle on the battlefield which has no limit of bullets. Basically who knocks off the opponent and survive in the end wins!

Play online!

Survival Shooter (2.5D Single-player Game)

Completed February 2016
Individual This is a Unity tutorial project from Unite training day 2014

Survival Shooter is a survival shooter game. The background story happens in the player's nightmares when all kinds of toys come to life and try to chase him. The scores are gained by how long you survive and how many enemies you knock off.

Play online!

Cube Run (2.5D Multi-player Game)

Completed December 2015, project at NTU Game Programming 2015
Team Poseidon, Kevin, Cliff, Vincent, Pushing

Cube Run is a Sci-Fi featured 2.5D multi-player game. Try to survive as long as possible by dodging any traps and unlocking various skills. Be sure not to hit your own track and other players.
• Implemented Multiplayer (port-to-port communication) function for the game
• Created and collected design materials for characters, traps, and scenes

Play online!
